**ToDoLy** — это современный, невероятно быстрый и визуально привлекательный органайзер задач прямо в вашем терминале (TUI). Написанный полностью на **Rust**, ToDoLy сочетает в себе классическую эстетику консольных приложений и мощный набор функций для управления временем, включая встроенного **полностью локального офлайн ИИ-помощника** на базе модели *SmolLM2-135M*.

---

## ✨ Основные особенности

* 🤖 **Встроенный локальный ИИ (SmolLM2-135M-Instruct):** Генерация списка из 5 конкретных подзадач для любой цели прямо в приложении. Работает 100% локально на CPU без отправки ваших данных в облако.
* ⌨️ **Двуязычная клавиатура:** Полноценная поддержка русской и английской раскладок — горячие клавиши работают корректно без необходимости переключения языка.
* 🎨 **8 встроенных тем оформления:** Мгновенное переключение цветовых палитр на лету (*Nordic Ice, Cyberpunk Magenta, Dracula Purple, Sunset Coral* и др.).
* 📊 **Интерактивный прогресс-бар:** Динамический расчёт выполненных дел на основе виджета `Gauge` от библиотеки Ratatui.
* 🔍 **Умный поиск и фильтрация:** Поиск по подстроке в режиме реального времени. Навигация адаптируется под отфильтрованный список.
* ♻️ **Надежная корзина (Recycle Bin):** Защита от случайного удаления с возможностью мгновенного восстановления или окончательного стирания (с подтверждением).
* 🏷️ **Система приоритетов:** Наглядная маркировка задач уровнями критичности (🟢, 🟡, 🟠, 🔴) с автоматическим перекрашиванием выполненных задач в серый цвет (`●`).
* 📁 **Легковесное сохранение:** Все данные пишутся в понятный текстовый формат в файлы `tasks.txt` и `trash.txt`.
* ⌨️ **Vim-навигация:** Поддержка перемещения по списку с помощью стрелочек или Vim-клавиш (`j` / `k`).

## 🚀 Быстрый старт

### Требования
* Установленный компилятор **Rust** (версия `1.80` или новее).

### Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ZAREZOV2/todoly-tui.git
   cd todoly-tui
   ```

2. Запустите приложение в режиме разработки:
   ```bash
   cargo run
   ```
   *Примечание: При первой сборке проект автоматически скачает квантованную модель ИИ (~90 МБ) и токенизатор (~2 МБ) с Hugging Face и вкомпилирует их прямо в бинарный файл.*

3. Сборка оптимизированного нативного `.exe` файла:
   ```bash
   cargo build --release
   ```
   Полученный автономный файл появится по пути `target/release/my_cli_tool.exe` (или `todoly-tui.exe`). Вы можете переносить его на любой ПК — он работает без установки рантаймов.

## ⌨️ Горячие клавиши (Управление)

| Клавиша (EN / RU) | Действие |
|:---|:---|
| ℹ️ **`F1`** | Показать / скрыть интерактивную справку (FAQ) |
| 🧑‍💻 **`F2`** | Показать окно «О программе» (информация о разработчике и статус автообновлений) |
| 🤖 **`i`** / **`ш`** | Открыть **ИИ-ассистента** для генерации подзадач по вашему промпту |
| ➕ **`a`** / **`ф`** | Добавить новую задачу в список |
| ✏️ **`e`** / **`у`** | Редактировать название выбранной задачи |
| ⚠️ **`Space`** / **`Enter`** | Отметить задачу как выполненную (или восстановить из Корзины) |
| ♻️ **`r`** / **`к`** | Переключить режим просмотра (Активные задачи / Корзина) |
| 🗑️ **`d`** / **`в`** | Переместить задачу в корзину (в Корзине — удалить навсегда с подтверждением) |
| 🧹 **`c`** / **`с`** | В списке: перенести все выполненные задачи в корзину. В корзине: полная очистка |
| 📶 **`p`** / **`з`** | Сменить приоритет задачи (Low -> Medium -> High -> Critical) |
| 🎨 **`t`** / **`е`** | Сменить цветовую тему оформления (8 тем по кругу) |
| ↕️ **`j` / `k`** (или `о` / `л`) | Перемещение вверх / вниз по списку |
| 🔄 **`Shift + ↑/↓`** / **`Shift + J/K`** | Переместить задачу выше/ниже в списке (изменить порядок) |
| 🔍 **`/`** / **`.`** | Активировать режим поиска по списку дел (Esc — сбросить поиск) |
| 🚪 **`q`** / **`й`** | Выход из программы |
